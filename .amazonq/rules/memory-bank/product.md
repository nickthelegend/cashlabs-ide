# Algorand IDE - Product Overview

## Purpose
A comprehensive web-based development environment for building, testing, and deploying Algorand smart contracts. The IDE provides an integrated experience for multiple Algorand development frameworks with built-in AI assistance, wallet integration, and real-time collaboration features.

## Key Features

### Multi-Framework Support
- **PyTeal**: Python-based smart contract development using PyTeal library
- **TealScript**: TypeScript-based smart contract development with AlgoKit
- **PuyaPy**: Pythonic smart contract development using AlgoPy framework
- **PuyaTs**: TypeScript version of PuyaPy for type-safe development

### Development Environment
- **Monaco Editor**: Full-featured code editor with syntax highlighting and IntelliSense
- **WebContainer Runtime**: In-browser execution environment for Node.js projects
- **Pyodide Integration**: Python execution directly in the browser
- **Real-time File System**: Live file updates with dual persistence (WebContainer + IndexedDB)
- **Multi-tab Editing**: Support for multiple open files with unsaved indicators

### Build & Deployment System
- **Integrated Terminal**: WebContainer-based terminal for build operations
- **Artifact Management**: Automatic discovery and management of compiled contracts
- **Smart Contract Deployment**: Direct deployment to Algorand TestNet/MainNet
- **Wallet Integration**: Built-in wallet creation and transaction signing
- **Network Support**: TestNet and MainNet connectivity with balance checking

### AI-Powered Assistant
- **RAG Implementation**: Retrieval-Augmented Generation with vector search
- **Template-Specific Knowledge**: Context-aware assistance for each framework
- **Code Generation**: AI-powered code suggestions and completions
- **Documentation Search**: Instant access to framework-specific documentation

### Project Management
- **Template System**: Pre-configured project templates for each framework
- **File Explorer**: Full CRUD operations on files and directories
- **Project Persistence**: Automatic saving with session recovery
- **Shareable Projects**: Public/private project sharing capabilities
